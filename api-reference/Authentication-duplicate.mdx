## Authentication

All API endpoints are authenticated using api keys. You can generate an API key on the [console](https://platform.shadeform.ai/settings/api). Example use below:

```bash
curl --request GET \
--url https://api.shadeform.ai/v1/instances/types \
--header 'X-API-KEY: <x-api-key>'
```
